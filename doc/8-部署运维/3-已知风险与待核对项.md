## 已知风险

| 风险 | 当前处理 |
| --- | --- |
| Docker 安装教程过于通用 | 本文不展开，按 Docker 官方文档安装 |
| 部分截图可能和当前 UI 不完全一致 | 只作为操作界面参考，不作为验证依据 |
| 前端环境变量 | 当前必须使用 `VITE_APP_*` |
| RocketMQ dashboard 未配置登录保护 | 生产环境不要公网暴露 |
| Seata 控制台 | 当前 Seata 2.6 compose 只使用 `8091` |
| Nacos 默认账号 | 本目录初始化 SQL 写入 `nacos / 80jpnH4.r5g`，而部分服务 bootstrap 默认值仍写着 `nacos/nacos`；如果启用 Nacos 鉴权，必须把两边统一 |
| 搜索同步 SQL | `update mall4cloud_product set update_time = now()`、`update mall4cloud_order set update_time = now()` 不匹配当前表名，不能照搬 |
| 支付回调地址 | `PayController` 当前仍硬编码网关地址，支付专题前不能把它写成生产可用配置 |
